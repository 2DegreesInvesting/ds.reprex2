### Public

* https://posit.cloud/ > New Project > New RStudio Project.
* Settings > Access > All posit cloud users.
* Work then share the link.

Example:
  
Install pak, reprex, datapasta, and "org/repo@pr"

```r
install.packages("pak")
pak::pak("reprex")
pak::pak("datapasta")
pak::pak("org/repo@pr")
```
